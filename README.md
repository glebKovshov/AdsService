# AdsService — сервис частных объявлений (WPF + SQL Server)

Учебный проект: информационная система для размещения и управления частными объявлениями (продажа/покупка/аренда/услуги).  
Реализованы просмотр объявлений без авторизации, регистрация/вход, личный кабинет пользователя, завершение объявлений с учётом прибыли, просмотр завершённых объявлений.

---

## Функциональные возможности

### Гость
- Просмотр всех объявлений
- Поиск по ключевым словам
- Фильтрация по:
  - городу
  - категории
  - типу объявления (продажа, покупка, аренда, услуги)
  - статусу (Активно, Завершено)

### Авторизованный пользователь
- Регистрация и авторизация (логин/пароль, пароль скрыт)
- Просмотр своих объявлений
- Добавление объявления
- Редактирование объявления (открывается по клику на элемент)
- Удаление объявления (с подтверждением)
- Изменение статуса объявления (Активно / Завершено)
- При завершении объявления вводится полученная сумма (ProfitAmount)
- Просмотр завершённых объявлений с указанием прибыли и итоговой суммы прибыли

---

## Технологии

- **WPF (.NET)** — пользовательский интерфейс
- **SQL Server** — база данных
- **ADO.NET Entity Framework (EDMX, Database First)** — доступ к данным
- **Frame Navigation** — навигация по страницам (Назад/Вперёд)
